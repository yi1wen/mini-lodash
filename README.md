## 项目介绍

本项目是一个基于 lodash 的 mini 版本，实现了 lodash 的核心方法，并且支持链式调用和惰性求值。

### 项目架构

1. 项目基于 lodash 实现，使用 webpack 打包，支持 ES6 模块。
2. 项目包含了 lodash 的核心函数，并且支持链式调用和惰性求值。
3. 项目使用 jest 进行测试，并且支持代码覆盖率报告。

### 核心函数

1. map
2. filter
3. reduce
4. curry
5. chain
6. debounce
7. throttle
8. cloneDeep
9. compose

### 使用方法

1. 安装依赖

```bash
npm install
```

2. 构建项目

```bash
npm run build
```

3. 测试

```bash
npm test
```
